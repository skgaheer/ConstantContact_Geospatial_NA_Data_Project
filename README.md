# ConstantContact_Geospatial_NA_Data_Project

This project included organizing Constituent data. One mission of the Council Member's office was to create stronger neighborhoods, which came in the form of creating neighborhood associations. 

What they had: A mailing list with important variables such as addresses and contact information. The office was manually going through street names and assigning them to already existing Neighborhood Associations. 

What I did:
  1. Created my own geospatial file of all existing Neighborhood Associations and potential future ones using Google Earth.
  2. Found a public dataset from the City of San Jose's Open Data Portal that had all the addresses in the City.
  3. Assigned each address in District 8 to a Neighborhood Association.
  4. Cross-referenced that to the original data (from Constant Contact) that the office was working with.
  5. Delivered a comprehensive list of all addresses in District 8, and the Neighborhood Association they belonged to, as well as the contact information.

Impact:
  1. Helped create 3 new Neighborhood Associations.
  2. Made a database of constituent data that would make outreach more targeted and efficient.
